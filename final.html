Shape Shade Madness |HTML/CSS| 2020

(Desktop Recommended if play) Story: Trying to escape a place where everyone who looks like you is the answer to discovering individually. The purpose of this game is that a particular shape that does not want to live as the other same shapes because the shape does not feel like themselves and wanted to explore it wanted to be. When I think of today’s trends like clothes or mainstream music, I noticed there’s no personality among people since they all like the same thing, so I wanted to explore what is unique about them from the others.

I expanded my code art game which is called, "Shape Shade Madness". I decided to create more challenging levels on the way to reach a reasonable ending. Therefore, I combined my Netart and Code Art Game for my final project. After completing each level there is a message of each shape's POV after their escape.

SJSU CVC Sticker Design

I used Canva to create a sticker for San Jose State University CVC dorm's merch for the upcoming fall semester. I chose the color red because it is the same color as the dorm's sticker on students' IDs despite the different shade of color.

It is hexagonal shape sticker in the color burgandy. Thw word "CVC" has the font block letter; it is seethrough with the whire borders around each letters. Finally, the palm tree symbolises as a calm and chilling vibe because the dorm is mostly quiet and private like living in paradise.
